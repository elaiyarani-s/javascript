Here are some important document attributes:

  • document.URL -- This is the actual URL of the website
  • document.body -- Everything in the body
  • document.head -- Everything in the head
  • document.links -- This is all the links on the page

Then we also have methods we can use to grab HTML elements

  • document.getElementById() -- Returns the element with the id
  • document.getElementsByClassName() -- Returns list of all elements belonging to a class
  • document.getElementsByTagName -- Returns list of all elements with tag
  • document.querySelector() -- Returns the first object matchinig the CSS style selector
  • document.querySelectorAlll() -- Returns all object matching the CSS style selector
